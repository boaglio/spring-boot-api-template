# Spring Boot API Template

Spring Boot API template app

## Why

This is an API REST based app ready for you to fork and code.

## Features

* Java 11 based
* Spring Boot 2.5 based
* Swagger ready 
* Simple API
* Simple Unit test
* Simple Home Page
* Commented code!
